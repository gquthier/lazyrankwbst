## 7. Pièges à éviter quand on automatise

- **Over-optimization** : densité > 3 % → pénalité Google. Garder < 1,5 %  
- **Duplicate** : vérifier Similarité < 8 % entre articles via Copyscape API  
- **Thin content** : objectif 1 000 mots minimum  
- **Over-publish** : > 10 URL/jour sur un jeune domaine → sandbox  
- **Neglect backlink** : même en automatisation, 2-3 liens qualité/mois = booster
